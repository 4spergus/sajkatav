# Agentic Pipeline (VS Code Extension)

VS Code extension for running the Agentic Pipeline through GitHub Copilot Chat.

Main command surface:
- `@pipeline /run <task>`
- `@pipeline /plan <task>`
- `@pipeline /spec <task>`
- `@pipeline /code <task>`
- `@pipeline /test <task>`

This package is bundled with esbuild and publishes only built output from `out/`.
